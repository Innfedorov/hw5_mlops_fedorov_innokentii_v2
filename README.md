# Домашнее задание 5. Обеспечение воспроизводимости эксперимента

## Цель проекта
Настроить воспроизводимый пайплайн машинного обучения с использованием DVC, MLflow и Feast.

## Как запустить
1. Установить зависимости: pip install -r requirements.txt
2. Для полной воспроизводимости требуется настроить DVC remote (Google Drive). Из-за ограничений Google OAuth в Colab это не выполнено. Локально пайплайн работает через dvc repro.

## Описание пайплайна
- prepare.py - делит датасет Iris на train/test
- train.py - обучает логистическую регрессию

## Где смотреть MLflow UI
mlflow ui --backend-store-uri sqlite:///mlflow.db

## Примечание
DVC remote не настроен из-за технических ограничений Google OAuth в Colab.
